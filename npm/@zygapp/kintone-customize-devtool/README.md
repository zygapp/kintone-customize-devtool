# @zygapp/kintone-customize-devtool

kintone カスタマイズ開発を Vite + HMR で行うための CLI ツール

## インストール

```bash
npm install -g @zygapp/kintone-customize-devtool
```

## 使い方

```bash
# プロジェクト初期化
kcdev init

# 開発サーバー起動
kcdev dev

# 本番ビルド
kcdev build

# kintone にデプロイ
kcdev deploy

# 依存パッケージを更新
kcdev update
```

## 対応プラットフォーム

- macOS (x64, ARM64)
- Linux (x64, ARM64)
- Windows (x64, ARM64)
